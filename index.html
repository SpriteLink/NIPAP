IPAM Love

This thing was built by network engineers for network engineers because nothing else out there matched our passiong for storing IP addresses. Lots of love built-in - guaranteed.

CLI

Comes with a feature complete CLI for anyone who prefers libreadline to a web browser.

Lightning fast

Never again wade through long list of IP addresses for finding the next available one. NIPAP's automagic address allocation functionality quickly gives you the next free address.

Python

Written in the world's most awesomest programming language. There's also a very convenient pynipap library for interfacing with NIPAP from Python.

Powerful search

Really great search engine built-in that does what you want. NIPAP understand IP prefixes and has search operators to match on more specific prefixes, covering prefixes and so on. Regular expressions are supported for everything else!

PostgreSQL

NIPAP uses the most advanced open source database on the planet to safely store your data. PostgreSQL offers native storage of IP addresses and super-fast ternary lookups using the ip4r module.

Equality

NIPAP treats all users, men and women, alike!

Java

Java client library for that true enterprise feeling.

Statistics

Integrated statistics helping you lie about your usage to your RIR!

IPv6

Do we even need to mention that IPv6 is supported? Naturally full feature parity with IPv4!

Reviews

NIPAP is the most powerful open-source solution in today's IPAM landscape. NIPAP is a rare and beautiful creature in that it has full feature parity between all its northbound components across all address-families, even in VRF context! The team's modern approach to software development and their focus on long-term manageability, give me the confidence to recommend NIPAP to everyone.

Job Snijders, Network Architect

We love NIPAP! We went from using a shared Google spreadsheet with very little standardization to an easy to use Web and CLI. Best of all the data is now in a uniform pattern and we can access it via a sane API. NIPAP FTW!

Augie Schwer of Safari Books Online

<+ytti> Axu, imma let the dude finish but http://spritelink.github.io/NIPAP/ is best IPAM of all time

ytti, network pro

DANTE operator of GEANT the pan European research and education network has been using NIPAP for some time now. DANTE selected NIPAP for its ability to provision subnets following predefined template for its ease of use and flexibility. NIPAP allows DANTE's teams to work from a single tool from the definition of the addressing template to its reservation and allocation ensuring a clean and well maintained address space. This cohesive approach is all the more critical when running both IPv4 and IPv6 natively which NIPAP help to maintain. It was all also critical for DANTE to select a tool able to work with Virtual routing instances (VRF). NIPAP is a very simple tool, very easy to put in place it allowed DANTE to maintain a well-organized address space very cheaply and very quickly. We would like to see the introduction of few features like the abilities to delegated part of the address space and the ability to create user groups with different responsibility of the address space. NIPAP is a "no muss, no fuss" tool very easy to deploy and DANTE will follow future development of the tool with great interest.

Sebastiano Buscaglione, DANTE
